# Ritesh Saini | Portfolio 🚀

**Live Preview:** [https://ritesh16-portfolio.vercel.app/](https://ritesh16-portfolio.vercel.app/)

A modern, highly-interactive developer portfolio built using React, Vite, and Tailwind CSS. The design focuses on glassmorphism aesthetics, fluid WebGL animations, and responsive clean layouts to showcase full-stack projects, experience, and AI capabilities.

## ✨ Features
- **Dynamic Splash Intro**: High-impact landing page with fluid background cursors and dynamic navigation grids.
- **Glassmorphism Design System**: Tailored dark-mode UI with sleek glowing borders and custom CSS utility animations.
- **Fully Responsive**: Flawless scaling from ultra-wide desktop monitors all the way down to mobile viewports.
- **Serverless Contact Form**: Web3Forms background API integration to securely deliver contact messages directly to email.

## 🛠️ Tech Stack
- **Core**: [React 19](https://react.dev/) + [Vite 6](https://vitejs.dev/)
- **CSS Framework**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animation Hub**: [Framer Motion](https://www.framer.com/motion/)
- **Iconography**: [Lucide React](https://lucide.dev/)

## 🚀 Quick Start
To get this project running locally on your machine:

1. **Clone the repository**
2. **Install Dependencies**
   ```bash
   npm install
   ```
3. **Start Development Server**
   ```bash
   npm run dev
   ```
   *The server will typically start on `http://localhost:5173`*

## 📦 Deployment
This app is compiled and fully optimized for seamless continuous deployment platforms like **Vercel**, **Netlify**, or **Cloudflare Pages**. 

When deploying, simply set the standard Vite build parameters:
- **Build Command**: `npm run build`
- **Output / Publish Directory**: `dist`
- **Node Version**: 18+ (or 20+)
